## Running the app

Before running the application one must install the dependencies:

```
#!shell

$ npm install
```

Then, run the script that creates points to be shown on the map:
```
#!shell

$ node src/migration/seed_points.js
```

To run the application type:

```
#!shell

$ npm start
```
